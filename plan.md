What are we building?
A community where users can upload their resumes and get critic from other users

Core Features:
1. User Authentication (JWT based)
2. Resume Upload
3. Resume preview on the website itself, and also a open in pdf view
4. a Homepage which can be accessed without login
5. need login to roast/upload resume
6. show all resumes on the homepage 
7. Users can Roast on resumes
8. Fire Metric for all resumes (users can give likes(fire) to resumes they like)
9. Profile dashboard which shows his uploaded resumes so he can manage them
10. A ranking system for resumes in the community

Cherry On Top features:
1. Users can toggle classified before uploading resume to remove private data/text from the resume
2. maybe a leaderboard or a top resumes page which have been liked the most / roasted the most


Expected Framework:
Frontend: React, TS, Vite
Backend: node, express, TS
DB: Supabase, Prisma
